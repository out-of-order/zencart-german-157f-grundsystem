Willkommen bei der deutschen Zen-Cart Version
Version 1.5.5f
vom 23.06.2018

Eine umfangreiche deutschsprachige Dokumentation und Installationsanleitung finden Sie im Ordner ANLEITUNG
Öffnen Sie die index.html im Ordner ANLEITUNG und lesen Sie vor einer Installation oder einem Update unbedingt die komplette Dokumentation.

Um Zen Cart 1.5.5f deutsch neu zu installieren nur die Ordner/Dateien im Ordner UPLOAD hochladen!

Um eine schlankere Installation und ein einfacheres Update zu ermöglichen, wurden die Demodaten ausgelagert.
Sie befinden sich im Ordner DEMODATEN

Ebenso ausgelagert wurden die Überprüfungstools im Ordner extras.
Sie befinden sich im Ordner TOOLS.

Wenn Sie bereits 1.5.5e deutsch verwenden und auf 1.5.5f deutsch updaten wollen verwenden Sie das spezielle Updatepaket von 1.5.5e auf 1.5.5f im Ordner UPDATE-VON-155e-AUF-155f
Die genauen Schritte dazu sind in der liesmich.txt in diesem Ordner beschrieben

Wenn Sie von einer älteren deutschen Zen Cart Version als 1.5.5e auf 1.5.5f deutsch updaten wollen, dann führen Sie ein normales Update durch so wie in der umfangreichen deutschsprachigen Dokumentation und Installationsanleitung im Ordner ANLEITUNG beschrieben.
